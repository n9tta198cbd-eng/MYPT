🚀 КАК ЗАПУСТИТЬ:

   1. Откройте файл index.html в любом браузере
      (двойной клик или ПКМ → Открыть с помощью → Браузер)

   2. Сайт готов к работе!

🎨 ЧТО НУЖНО ИЗМЕНИТЬ:

   ✏️  ПЕРСОНАЛЬНАЯ ИНФОРМАЦИЯ (в index.html):
       - Контактные данные в hero-top-bar (строка 38-41)
       - Ваше имя, email, телефон, город в секции "О себе"
       - Текст "О себе"
       - Описание проектов
       - Подпись в hero-signature (строка 59)

   📸 ИЗОБРАЖЕНИЯ:
       Замените placeholder изображения на свои в папке images/:
       - profile.jpg (500x500px) - ваше фото
       - project1-6.jpg (800x600px) - скриншоты работ

   🎨 ЦВЕТА (в styles.css, строка 8-17):
       Измените CSS переменные для другой цветовой схемы

🌐 ПУБЛИКАЦИЯ САЙТА:

   Бесплатные варианты:

   ⭐ GitHub Pages (рекомендуется):
      1. Создайте репозиторий на github.com
      2. Загрузите все файлы
      3. Settings → Pages → выберите ветку main
      4. Готово! Сайт по адресу: username.github.io/repo-name

   ⭐ Netlify (netlify.com):
      Просто перетащите папку с проектом на сайт

   ⭐ Vercel (vercel.com):
      Импортируйте проект из GitHub
