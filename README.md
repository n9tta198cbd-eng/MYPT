# Сайт-портфолио мультидисциплинарного дизайнера

Современный, адаптивный сайт-портфолио с анимированным процедурным фоном в стиле технических схем и крупной выразительной типографикой.

## Особенности

- **Уникальный анимированный фон** - процедурная генерация технических элементов на Canvas
- **Крупная типографика** с эффектом text-stroke и тенями
- **Минималистичный дизайн** в стиле n9tta
- **Полная адаптивность** для всех устройств
- **Интерактивная фильтрация** портфолио по категориям
- **Плавные анимации** при прокрутке
- Секции: Главная, О себе, Портфолио (2D/3D/Графика), Навыки, Контакты

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (Canvas API)
- Google Fonts (Playfair Display, Poppins)

## Установка и запуск

1. Скачайте все файлы проекта
2. Добавьте свои изображения в папку `images/`:
   - `profile.jpg` - ваше фото (рекомендуемый размер: 500x500px)
   - `project1.jpg` до `project6.jpg` - изображения ваших работ (рекомендуемый размер: 800x600px)
3. Откройте файл `index.html` в браузере

## Настройка контента

### Изменение персональной информации

Откройте `index.html` и найдите секцию "About" (строка ~48):

```html
<div class="info-item">
    <span class="info-label">Имя:</span>
    <span class="info-value">Ваше Имя</span>
</div>
```

Замените "Ваше Имя" на свои данные.

### Изменение проектов в портфолио

Найдите секцию "Portfolio" (строка ~110) и измените:
- Заголовок проекта (`<h3>`)
- Категорию (`<p>`)
- Атрибут `data-category` для фильтрации

### Изменение цветовой схемы

Откройте `styles.css` и измените CSS-переменные в начале файла:

```css
:root {
    --primary-color: #8e44ad;
    --secondary-color: #3498db;
    --accent-color: #e74c3c;
    /* ... */
}
```

## Публикация сайта

### GitHub Pages (бесплатно)

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. Перейдите в Settings → Pages
4. Выберите ветку `main` и папку `root`
5. Сохраните - ваш сайт будет доступен по адресу: `https://ваш-username.github.io/название-репозитория/`

### Netlify (бесплатно)

1. Зарегистрируйтесь на [netlify.com](https://www.netlify.com)
2. Перетащите папку с проектом в Netlify Drop
3. Ваш сайт будет опубликован автоматически

### Vercel (бесплатно)

1. Зарегистрируйтесь на [vercel.com](https://vercel.com)
2. Импортируйте проект из GitHub или загрузите напрямую
3. Сайт будет развернут автоматически

## Дополнительные возможности

### Интеграция формы обратной связи

Для работы формы обратной связи используйте один из сервисов:

1. **Formspree** ([formspree.io](https://formspree.io))
   - Замените атрибут `action` формы на URL от Formspree

2. **EmailJS** ([emailjs.com](https://www.emailjs.com))
   - Добавьте их SDK и настройте отправку email

3. **GetForm** ([getform.io](https://getform.io))
   - Простая интеграция через URL
